# Face Mask Detection Web App (Customised)

This project provides a simple web application that detects whether a face
in an uploaded image is wearing a mask. It uses a synthetic dataset to
train a lightweight machine‑learning model and serves predictions via a
FastAPI endpoint. A minimal HTML front‑end allows users to upload images
and view the results.

The goal of this customised version is to demonstrate end‑to‑end
development—from generating data and training a model to deploying a
web service—without relying on heavy deep‑learning frameworks. It is
inspired by more complex face mask detection projects【647200950190949†L31-L39】 but uses
scikit‑learn and OpenCV to keep things simple and easy to run on
resource‑constrained machines.

## Features

* **Synthetic dataset:** A script automatically generates a dataset of
  200 images (100 with masks and 100 without) using simple geometric
  shapes. This avoids any copyright issues and makes the project
  self‑contained.
* **Model training:** The training script reads the images, flattens
  them into feature vectors and trains a Support Vector Machine (SVM)
  classifier. The trained model is saved as `models/mask_detector.pkl`.
* **FastAPI service:** The `app.py` script loads the model and exposes
  two endpoints: `/health` for a quick health check and `/predict`
  for image classification. It accepts image uploads, processes them
  with OpenCV and returns a JSON response indicating whether a mask
  was detected.
* **Web interface:** A simple `frontend/index.html` page lets users
  upload an image and displays the prediction returned by the API.
* **Docker‑friendly:** All dependencies are specified in `requirements.txt`.
  You can run the app locally or build it into a Docker container.

## Setup and usage

### 1. Install dependencies

Use a virtual environment to install the requirements:

```bash
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

### 2. Generate the dataset

The repository includes a script to create a synthetic dataset. It uses
OpenCV to draw simple faces with and without masks:

```bash
python scripts/generate_dataset.py --output-dir dataset --num-samples 100
```

This will create two subfolders `with_mask` and `without_mask`, each
containing `num_samples` PNG images.

### 3. Train the model

Train an SVM classifier on the generated dataset:

```bash
python train.py --data-dir dataset --model-out models/mask_detector.pkl
```

The script will report the training and test accuracy and save the model.

### 4. Run the web service

Start the FastAPI server:

```bash
uvicorn app:app --reload --port 8000
```

Navigate to `frontend/index.html` in your browser (e.g. via `python -m
http.server --directory frontend 8001`) and upload an image of a face.
The page will display whether a mask was detected.
